Générer sa signature

Il suffit de remplir le formulaire avec les informations demandées.

Le nom, le prénom et la fonction peuvent être écrits en minuscule. Ils seront reformatés pour s'afficher joliment.

La ligne "Téléphone" se remplit sous la forme "Tel : +33 (0)4 93 16 78 60"
Il est important de bien mettre l'indicatif Fr (+33) et de séparer les chiffres par paire séparées par une espace.
Afin de bien respecter la norme d'affichage, séparer les différents numéros de téléphone par un " - "

Appuyer ensuite sur le bouton vert "Générer la signature".
Le navigateur construit le fichier signature et l'enregistre automatiquement dans votre dossier "Téléchargements" avec le nom "signature_prenom_nom.htm".

Ouvrir le fichier reçu dans votre navigateur, copier son contenu (en tapant CTRL A puis CTRL C) avant de le coller dans outlook (CTRL V dans la section appropriée des paramètres outlook).

Comment ajouter sa signature dans Outlook pour le web

  1. Ouvrir Outlook pour le web.
  2. Cliquer sur les paramètres (en haut à droite de la fenêtre).
  3. Dans la fenetre qui s'ouvre, cliquer sur Compte (colonne de gauche) puis sur Signatures.
  4. Cliquer sur nouvelle signature et coller (CTRL V) dans la zone d'édition la signature générée précédemment. La signature sera également disponible dans le client Outlook au prochain démarrage de l'application.
Lors de l'édition d'un message, la sélection de la signature est possible par le menu Insérer / Signature.

Comment ajouter sa signature dans Outlook mobile

  1. Ouvrir Outlook Mobile
  2. Accéder aux paramètres: cliquer en haut à gauche (maison ou initiales du compte sélectionné) pour faire apparaître le bandeau latéral, puis sélectionner les paramètres en bas du bandeau latéral.
  3. Cliquer sur Signatures
  4. Coller la signature précédemment générée, pour le compte voulu (ou pour tous les comptes).
