Générer sa signature

Il suffit de remplir le formulaire avec les informations demandées.

Le nom, le prénom et la fonction peuvent être écrits en minuscule. Ils seront reformatés pour s'afficher joliment.

La ligne "Téléphone" se remplit sous la forme "Tel : +33 (0)4 93 16 78 60"
Il est important de bien mettre l'indicatif Fr (+33) et de séparer les chiffres par paire séparées par une espace.
Afin de bien respecter la norme d'affichage, séparer les différents numéros de téléphone par un " - "

Appuyer ensuite sur le bouton vert "Générer la signature".
Le navigateur construit le fichier signature et l'enregistre automatiquement dans votre dossier "Téléchargements" avec le nom "signature_prenom_nom.htm".

Pour finir, déplacer le fichier reçu du dossier "Téléchargements" vers le dossier "Documents" en le faisant glisser d'un répertoire à l'autre.
Ne surtout pas utiliser le navigateur web pour cette dernière opération, car le navigateur web modifie les fichiers qu'il enregistre et copie les logos sur votre disque, ce qui les fait disparaître de votre signature mail reçue par vos correspondants.

Comment ajouter sa signature à Thunderbird

  1. Cliquer sur votre adresse e-mail (au dessus du dossier "Courrier entrant").
  2. Sur la page affichée, cliquer sur "Paramètres du compte" en haut à droite.
  3. Dans la fenetre qui s'ouvre, cocher la case "Apposer la signature à partir d'un fichier, puis sélectionner le fichier que vous venez d'enregistrer dans votre répertoire "Documents".

Comment ajouter sa signature à Zimbra

  1. Dans Zimbra :
    1. Cliquez sur l'onglet "Préférences", puis dans le menu de gauche, sur "Mail".
    2. Dans la fenêtre qui apparait, sur la droite, cochez l'option de création des mails au format HTML.
    3. Cliquez sur l'onglet "Préférences", puis dans le menu de gauche, sur "Signatures"
    4. Dans la fenêtre qui apparait, sur la droite, cliquez sur le bouton "format texte simple" et sélectionnez "format HTML"
    5. Si vous aviez une signature déja configurée, supprimez la : cliquez dans la zone de saisie puis ctrl-A, puis appuyez sur le bouton Suppr du clavier
  2. Ouvrez dans votre navigateur web votre fichier signature (double clic sur le fichier que vous avez enregistré précédemment dans le répertoire "Documents" de votre PC).
  3. Votre signature s'affiche dans une nouvelle page du navigateur web. Sélectionnez tout le contenu de la page (ctrl-A), copiez (ctrl-C) et collez dans la zone de texte de zimbra (ctrl-V), en supprimant l'existant si nécessaire. Réglez l'utilisation des signatures, puis, pour valider les modifications, appuyer sur "Enregistrer".
